# jafa.dev — Jorge de la Flor

Sitio web personal de Jorge de la Flor, Software Architect & Protocol Developer.

## Tecnologías

- Jekyll
- SCSS
- GitHub Pages

## Desarrollo local

```bash
bundle install
bundle exec jekyll serve
```

## Estructura

- _config.yml — Configuración del sitio
- index.md — Página principal
- _posts/ — Entradas del blog
- md_pages/ — Páginas estáticas (About, Services, Projects, Stack, Posts, Contact)

## Despliegue

El sitio se despliega automáticamente en GitHub Pages desde la rama `main`.
