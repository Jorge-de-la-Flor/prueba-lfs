## 2026

### 🎤 Microsoft Build 2026 Community Event — Azure User Group Latam
*📅 IDAT Lima, Perú · Junio 2026*

**"Building a Multi-Tenant Python Runtime on Azure Functions"**

Ponencia invitada donde compartí la arquitectura y lecciones aprendidas al desarrollar **Apider**, un runtime serverless multi-tenant con:
- Aislamiento por ContextVar
- Cifrado Fernet de credenciales en tránsito
- Servidor MCP stateless para integración con agentes IA
- Facturación Paddle con control de excesos
- 61 pruebas de integración en CI

> *"Fue una experiencia increíble compartir con la comunidad latinoamericana de Azure y mostrar cómo Python y Rust pueden trabajar juntos en la nube."*
